## Specifications

| Specification | Value |
|--------------|-------|
| CPU | TMS9900 @ 3.0 MHz |
| Graphics | TMS9918A VDP |
| Sound | TMS9919 (SN76489) |
| RAM | 16 KB (256 bytes "scratchpad" + 16 KB VDP RAM) |
| VRAM | 16 KB (dedicated to VDP) |
| ROM | 26 KB (8 KB BIOS + 18 KB BASIC) |
| Display | 32x24 text, 256x192 graphics (16 colors) |
| Storage | Cassette, optional floppy |
| Cartridge | ROM cartridge slot |
| Expansion | Peripheral Expansion Box (PEB) |

## Features

- True 16-bit CPU (first 16-bit home computer)
- TI BASIC built-in
- Solid-state cartridge software
- Speech synthesizer support
- Expansion via Peripheral Expansion Box

## Peripherals

- Peripheral Expansion Box (PEB)
- Speech Synthesizer
- Disk Controller + Floppy Drives
- RS-232 Interface
- Memory Expansion (32 KB)
- TI Joysticks

## Notes

The TI-99/4A was the first 16-bit home computer, though its architecture bottlenecked CPU-to-RAM access through the VDP. The speech synthesizer was a notable accessory. TI exited the home computer market in 1983.
