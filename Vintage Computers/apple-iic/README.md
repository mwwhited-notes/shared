|--------------|-------|

| Display | 40/80 column text, 280x192 graphics |
| Storage | Internal 5.25" floppy (140 KB) |
| Ports | 2x Serial (DB-19), Video, Joystick, Disk |

## Features

- Built-in 5.25" floppy drive
- Built-in 80-column display support
- Built-in serial ports (modem and printer)
- Portable design with carrying handle
- No internal expansion slots (compact design)
- Compatible with most Apple II software

- Apple IIc Monitor (9" green/amber phosphor)
- External 5.25" floppy drive (Disk IIc)
- UniDisk 3.5 (800 KB 3.5" drive)
- RAM expansion (up to 1 MB)
- Mouse

| Specification | Value |
|--------------|-------|
| Manufacturer | Big Mess O' Wires |
| Model | Floppy Emu Model C |
| Storage | SD Card |
| Emulation | 5.25" floppy, 3.5" floppy, SmartPort HD |

Features:
- Emulates 5.25" (140K) and 3.5" (800K) floppy drives
- SmartPort hard disk emulation (up to 4x 2GB volumes)
- Supports .DSK, .PO, .DO, .2MG, .NIB, .WOZ formats

- No drivers required - behaves like real hardware

## Software

- ProDOS (preferred operating system)
- Apple DOS 3.3
- AppleWorks

## Notes

The IIc was Apple's first "portable" computer (not battery-powered, but compact). Unlike the Apple IIe, it has no internal expansion slots - all I/O is built-in.
