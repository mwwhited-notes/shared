# Apple IIgs

**16-bit Apple II with Graphics and Sound**

## Specifications

| Specification | Value |
|--------------|-------|
| CPU | WDC 65C816 @ 2.8 MHz |
| RAM | 256 KB - 8 MB (expandable) |
| ROM | 128 KB / 256 KB (ROM 01/03) |
| Display | 320x200 / 640x200, 16/256 colors |
| Graphics | Custom VGC (Video Graphics Controller) |
| Sound | Ensoniq DOC (32 oscillators, 8-bit) |
| Storage | 3.5" (800 KB) and 5.25" floppy |
| Ports | ADB, Serial, Disk, Video, Audio |
| Expansion | 7 Apple II slots + Memory expansion |
| Power | Internal power supply |

## Features

- 16-bit processor (backward compatible with 8-bit Apple II)
- Ensoniq wavetable synthesizer (32 voices)
- Super hi-res graphics (3200 colors possible)
- Apple Desktop Bus (ADB) keyboard/mouse
- GS/OS - GUI operating system
- Full Apple II compatibility mode

### Floppy Emu (shared with IIc)

| Specification | Value |
|--------------|-------|
| Manufacturer | Big Mess O' Wires |
| Model | Floppy Emu Model C |
| Storage | SD Card |
| Emulation | 5.25" floppy, 3.5" floppy, SmartPort HD |

Features:
- Emulates 5.25" (140K) and 3.5" (800K/1.4MB) floppy drives
- SmartPort hard disk emulation (up to 4x 2GB volumes)
- Supports .DSK, .PO, .DO, .2MG, .NIB, .WOZ, .MOOF formats
- On-screen menu for disk selection
- No drivers required - behaves like real hardware

- HyperCard IIgs
- Platinum Paint, DeluxePaint

## Notes

The IIgs was the most powerful Apple II, featuring a 16-bit CPU and the same sound chip used in the Mirage and ESQ-1 synthesizers. It was the last Apple II model produced (discontinued 1992).
