## Organization

The collection is organized into 12 main directories following a structured learning path:

```
analog-computing-repository/
├── 01-fundamentals/         # Start here: Core concepts and principles
├── 02-components/           # Electronic components used in analog computing
├── 03-mathematics/          # Mathematical foundations and theory
├── 04-applications/         # Real-world applications and case studies
├── 05-historical/           # Vintage computer systems and papers
├── 06-modern-systems/       # Contemporary analog computing projects
├── 07-tutorials/            # Step-by-step guides and experiments
├── 08-references/           # Lookup materials and indices
├── 09-datasheets/pdfs/      # Component datasheets (159 PDFs)
├── 10-application-notes/pdfs/  # Application notes from IC manufacturers
├── 11-aerospace-simulation/ # Apollo, fire control, hybrid computing (53 PDFs)
└── 12-analog-computer-docs/ # Vintage manuals (21 PDFs)
```

## Notable Collections

### Aerospace Simulation (53 PDFs)

Contains documentation of flight simulators and control systems from Apollo program, fire control systems, and hybrid computing applications.

**Key materials:**
- Apollo guidance computer documentation
- Digital-analog hybrid systems for aerospace
- Fire control system algorithms
- Historical flight simulator designs

### Datasheets & Application Notes (159+ PDFs)

Complete collection of IC datasheets from Analog Devices, Texas Instruments, and other manufacturers including:

- Op-amp datasheets (various technologies)
- Precision amplifier circuits
- ADC/DAC application notes
- Signal conditioning circuits
- Power supply designs

### Vintage Computer Manuals (21 PDFs)

Documentation for classic analog computer systems:

- EAI analog computers
- Heathkit analog computer kits
- Comdyna analog systems
- Hybrid computer systems
