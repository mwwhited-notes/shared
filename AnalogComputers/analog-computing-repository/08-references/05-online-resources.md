## Simulation Tools

### Free Simulators

**LTspice**
- https://www.analog.com/en/design-center/design-tools-and-calculators/ltspice-simulator.html
- Industry-standard free simulator
- Extensive component library
- Highly recommended

**TINA-TI**
- https://www.ti.com/tool/TINA-TI
- Free from Texas Instruments
- SPICE-based simulation

**Falstad Circuit Simulator**
- https://falstad.com/circuit/
- Browser-based
- Visual, intuitive
- Good for learning

**Qucs (Quite Universal Circuit Simulator)**
- http://qucs.sourceforge.net/
- Open source
- Multiple simulation types

**CircuitLab**
- https://www.circuitlab.com/
- Web-based
- Free tier available

### Commercial Simulators

**Multisim** (National Instruments)
- Educational version available
- Comprehensive component library

**PSpice** (Cadence)
- Industry standard
- Student versions

**Simulink** (MathWorks)
- Block diagram simulation
- Good for systems modeling

## Summary: Getting Started

1. **Learn basics:** All About Circuits, Electronics Tutorials
2. **Simulate:** LTspice (free, excellent)
3. **Watch:** EEVblog, Ben Eater videos
4. **Ask:** EEVblog forum, Stack Exchange
5. **Build:** Get components from Digi-Key/Mouser
6. **Explore:** THE Analog Thing for hands-on learning
