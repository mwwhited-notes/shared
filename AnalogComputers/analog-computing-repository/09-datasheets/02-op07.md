# OP07 Precision Operational Amplifier

## Overview

The OP07 is an ultralow offset voltage operational amplifier providing extremely low input offset voltage (75 μV max) and very low offset drift (1.3 μV/°C typ). It is designed for precision instrumentation and analog computation where DC accuracy is critical.

## Features

- Ultra-Low Offset Voltage: 75 μV max (OP07E grade)
- Low Offset Voltage Drift: 1.3 μV/°C typ
- Excellent Long-Term Stability: 0.4 μV/month
- Low Input Bias Current: 2 nA typ
- High Open Loop Gain: 400 V/mV typ
- High CMRR: 110 dB typ
- Wide Supply Range: ±3V to ±18V
- No External Components Required

## Error Analysis for Integrators

### Drift Sources

| Source | Contribution | Mitigation |
|--------|--------------|------------|
| Vos (offset) | -(Vos/RC) × t | Use offset trim |
| Ib (bias current) | Ib/C × t | Use FET input or compensation |
| Capacitor leakage | IC leakage/C × t | Use high-quality caps |
| Temperature | Vos drift | Temperature control |

### Drift Calculation Example

For τ = 1s integrator with OP07:
- Vos drift: 60μV / 1s = 60 μV/s = 0.216 V/hour
- With compensation resistor: < 50 mV/hour achievable

## Best Practices

1. **Decoupling**: Use 0.1μF ceramic + 10μF electrolytic on each supply
2. **Grounding**: Use star ground, separate analog/digital grounds
3. **Capacitor Selection**: Polypropylene or polystyrene for integrators
4. **Thermal Considerations**: Allow 15-30 min warm-up for best accuracy
5. **PCB Layout**: Keep traces short, use guard rings for high-impedance nodes
