# AD633 Analog Multiplier

## Overview

The AD633 is a functionally complete, four-quadrant, analog multiplier. It includes high impedance differential X and Y inputs, and a high impedance summing input (Z). The low impedance output voltage is a nominal 10V full scale provided by a buried Zener reference.

## Transfer Function

```
W = (X1 - X2)(Y1 - Y2) / 10V + Z
```

Where:
- X1, X2: Differential X inputs
- Y1, Y2: Differential Y inputs
- Z: Summing input
- W: Output
- 10V: Scale factor (internal reference)

```
       ┌──────────────┐

       └──────────────┘
```

## Basic Configurations

### Simple Multiplication

```
X ──── X1     W ──── Output = XY/10
       X2 ─┬
            │
Y ──── Y1   │
       Y2 ─┬│
            ││
Z ─────────┴┴─ GND

Output: W = XY/10
```

### Squaring Circuit

```
X ──┬── X1     W ──── Output = X²/10
    │   X2 ─── GND
    │
    └── Y1
        Y2 ─── GND
        Z ──── GND

Output: W = X²/10
```

### Division (Feedback Configuration)

```
                    ┌─────────────────┐
                    │                 │
Numerator (N) ── X1 │                 │
                 X2 ┴ GND             │
                    │      AD633      │
          ┌──── Y1 ─┤                 ├── W = 10N/D
          │      Y2 ┴ GND             │
          │         │                 │
          │  Offset ┴ Z               │
          │         │                 │
          │         └─────────────────┘
          │                 │
          └─────────────────┘
            (Feedback)

Denominator (D) applied to Y via feedback
Output: W = 10N/D (for D > 0)
```

### Square Root

```
            ┌─────────────────────────┐
            │                         │
Input ── X1 │                         │
         X2 ┴ GND                     │
            │        AD633            │
    ┌─── Y1 ┤                         ├── W = √(10×Input)
    │    Y2 ┴ GND                     │
    │       │                         │
    │    Z ─┤                         │
    │       └─────────────────────────┘
    │                   │
    └───────────────────┘

Output: W = √(10 × Input)
```

## Analog Computing Applications

### Nonlinear Equation: xy Term

Used in Lorenz attractor: dy/dt = x(ρ - z) - y

```
x ──── X1
       X2 ─── GND            W = x(ρ-z)/10

(ρ-z) ─ Y1
        Y2 ─── GND
        Z ──── GND
```

### Multiplication in Van der Pol

For term: μ(1 - x²)·x'

```
Stage 1: Compute x²
x ──┬── X1     W1 = x²/10
    └── Y1

Stage 2: Compute (1 - x²)·x'
Use summer to create (1 - x²), then multiply by x'
```

### Modulation

```
Carrier ── X1
           X2 ─── GND
                            W = Carrier × (1 + m·Signal)/10
Signal ─── Y1
    ┌───── Y2
    │      Z ──── GND
    │
DC Bias for 1+m·Signal
```

## Error Sources and Mitigation

| Error Source | Magnitude | Mitigation |
|--------------|-----------|------------|
| Scale factor | ±1% | Calibration pot |
| Offset | ±50 mV | Trim at Z input |
| Nonlinearity | ±1% | Use better multiplier (AD534) |
| Temperature | 0.02%/°C | Temperature control |
| Feedthrough | ±1% | Input balance adjustment |
