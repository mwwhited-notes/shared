### Comdyna GP-6
Popular desktop analog computer for education and research. Over 2000 units installed worldwide.

| comdyna-gp6-operators.pdf | GP-6 operator's manual |

**Specifications:**
- Self-contained unit for systems up to 4 state variables
- ±10V computing range
- Designed for dynamic system simulation
