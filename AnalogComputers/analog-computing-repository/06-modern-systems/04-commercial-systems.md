# Commercial Analog Computing Systems

## Overview

While the golden age of large analog computers has passed, several commercial systems remain available for education, research, and specialized applications.

## Currently Available Systems

### THE Analog Thing (THAT)

**Manufacturer:** anabrid GmbH (Germany)
**Website:** [the-analog-thing.org](https://the-analog-thing.org/)

**Specifications:**
- 5 integrators, 4 summers, 2 multipliers
- Portable, open-source design
- ~$300-350

**Best for:** Education, hobbyists, learning

See [detailed THAT documentation](01-the-analog-thing.md)

### Analog Paradigm Model-1

**Manufacturer:** Analog Paradigm (Germany)
**Website:** [analogparadigm.com](https://analogparadigm.com/)

**Specifications:**
- Modular system
- Professional-grade components
- Expandable architecture
- Higher precision than THAT

**Best for:** Research, advanced education, professional use

### Comdyna GP-6

**Manufacturer:** Comdyna (USA - legacy)

**Status:** May still be available used/refurbished

**Specifications:**
- 6 integrators
- Desktop format
- Educational design

**Best for:** Classroom instruction

### Restoration Considerations

- Vacuum tube systems require tube replacement
- Capacitors often need replacing (aging)
- Calibration required
- Documentation availability varies

## Building vs. Buying

### Advantages of Commercial Systems

1. **Tested and calibrated**
2. **Documentation provided**
3. **Technical support (for current products)**
4. **Known specifications**
5. **Ready to use**

### Advantages of DIY

1. **Lower cost (potentially)**
2. **Deep learning through building**
3. **Customizable**
4. **Maintainable (you understand it)**
5. **Satisfaction of creation**

### Recommendations by Use Case

| Use Case                | Recommendation      |
|-------------------------|---------------------|
| Learning basics         | DIY simple circuits |
| Education (classroom)   | THAT or Comdyna     |
| Research (moderate)     | THAT + expansion    |
| Research (advanced)     | Analog Paradigm     | 
| Professional simulation | Custom or hybrid    |
| AI acceleration         | Mythic, Rain, etc.  |

## Procurement Considerations

### For Educational Institutions

1. **Budget considerations**
   - THAT: ~$300/unit
   - Class set: ~$3000-5000

2. **Curriculum integration**
   - Documentation quality
   - Exercise availability
   - Technical support

3. **Durability**
   - Student-proof design
   - Replacement parts availability

### For Research

1. **Performance requirements**
   - Precision needed
   - Speed requirements
   - Integration capabilities

2. **Expandability**
   - Modular design
   - Hybrid interface

3. **Support**
   - Technical documentation
   - Manufacturer responsiveness
