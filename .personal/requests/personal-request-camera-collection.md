# Personal Request: Camera Collection Inventory

## What We Need

Current inventory of cameras, lenses, and photography accessories from Personal/Camera Collection.

## Use Case

While no active projects currently reference the camera collection, this information is useful for:
- Photography project planning and equipment compatibility
- Lens mount compatibility tracking (M42, Canon EF, Nikon F, etc.)
- Planning photography-focused projects
- Documenting vintage and specialty glass
- Cross-referencing equipment with film/digital capabilities
- Planning photography workflow automation projects

## Information to Extract From Personal/Camera Collection/

For each camera body, lens, and accessory, provide:

### Camera Bodies
- Camera model and brand
- Camera type (Digital/Film/Hybrid)
- Sensor size / Film format
- Megapixels (for digital) or film speed range
- Lens mount type
- Key features (autofocus, image stabilization, etc.)
- Current status (Active/Archive/For Repair/Collecting)
- Condition and serial number tracking
- Compatible batteries and accessories

### Lenses
- Lens model and brand
- Focal length and aperture range
- Mount type
- Special features (image stabilization, macro capability, etc.)
- Condition and status
- Compatibility notes (full-frame, crop-sensor, vintage adapters)
- Accessory compatibility (filters, teleconverters, extension tubes)

### Accessories & Support Equipment
- Type (tripod, flash, filters, cases, adapters, etc.)
- Model and specifications
- Compatibility with camera systems
- Status and condition

### Categories Needed

1. **Digital Cameras** - DSLR, mirrorless, compact digital
   - Model
   - Sensor size/MP
   - Mount type
   - Status

2. **Film Cameras** - SLR, rangefinder, compact film
   - Model
   - Film format
   - Mount type
   - Status

3. **Vintage/Specialty Cameras** - Collectible, rare, or specialty purpose
   - Model
   - Type and era
   - Status

4. **Lenses** - By mount type (Canon EF, Nikon F, M42, Sony E, etc.)
   - Focal length
   - Aperture range
   - Condition

5. **Teleconverters & Adapters** - Mount converters, tube rings, macro adapters
   - Type and ratio
   - Compatible mounts
   - Status

6. **Flashes & Lighting** - External flashes, light meters, lighting equipment
   - Model and type
   - Compatibility
   - Status

7. **Tripods & Support** - Tripods, monopods, stabilization equipment
   - Model
   - Compatibility
   - Status

8. **Cases & Storage** - Camera bags, lens cases, environmental protection
   - Model
   - Capacity
   - Status

9. **Filters & Lens Accessories** - UV filters, polarizers, ND filters, macro rings
   - Type and size
   - Compatibility
   - Quantity
