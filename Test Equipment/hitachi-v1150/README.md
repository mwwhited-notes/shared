# Hitachi V-1150

**Analog Oscilloscope**

## Specifications

| Specification | Value |
|--------------|-------|
| Bandwidth | 150 MHz |
| Channels | 4 |
| Display | 6" CRT, internal graticule (8×10 div) |
| Max Input | 400 V DC + peak AC (at 1 kHz) |

## Features

- Dual timebase
- Delayed sweep
- X-Y mode
- Quad mode with independent positioning
- CMRR ≥20 dB at 20 MHz
