# Rigol DS1102E

**Digital Storage Oscilloscope**

## Specifications

| Specification | Value |
|--------------|-------|
| Bandwidth | 100 MHz |
| Channels | 2 + external trigger |
| Sample Rate | 1 GSa/s real-time, 25 GSa/s equivalent |
| Memory | 1 Mpts |
| Resolution | 8-bit |
| Display | 5.6" 64K color TFT |

## Features

- Edge/pulse/video/slope/pattern triggers
- FFT analysis
- Pass/fail testing
- Waveform record/replay
- 20 auto measurements
- USB host/device
