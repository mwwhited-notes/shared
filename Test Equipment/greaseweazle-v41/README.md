| Format | Description |
|--------|-------------|
| ADF | Amiga Disk File |
| IPF | Interchangeable Preservation Format |
| DSK | Various systems |
| IMG | Raw sector image |
| HFE | HxC Floppy Emulator |
| SCP | SuperCard Pro (flux-level) |

## Features

- Raw flux transition capture (preserves copy protection)
- Write support for most formats
- Cross-platform software (Windows, Linux, macOS)
- Open source firmware and software
- Factory tested before shipping

## Software

- **Greaseweazle Tools** - Command-line utilities
- **FluxEngine** - Compatible alternative
- **HxC Floppy Emulator Software** - GUI option

## Applications

- Vintage floppy disk preservation
- Creating working disks for retro computers
- Recovering data from old media
- Analyzing copy protection schemes
- Transferring disk images to physical media

## Requirements

- Compatible floppy drive
- Floppy ribbon cable (with twist for PC drives)
- Power supply for drive (or use integrated 3.5" power)

## Notes

The Greaseweazle captures raw magnetic flux transitions rather than decoded data, allowing preservation of copy-protected disks and unusual formats that standard controllers cannot read.
