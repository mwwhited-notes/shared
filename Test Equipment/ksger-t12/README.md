## Features

- PID temperature control
- OLED display with real-time temperature
- 5 temperature presets
- Per-tip calibration support
- Auto-sleep function
- Quick heating (~8 seconds to working temp)
- Thermocouple cold junction compensation (-9 to 99°C)
