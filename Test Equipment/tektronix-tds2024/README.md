# Tektronix TDS2024 + TDS2CMA Module

**Digital Storage Oscilloscope with Communications Module**

## Specifications

| Specification | Value |
|--------------|-------|
| Bandwidth | 200 MHz |
| Channels | 4 analog |
| Sample Rate | 1 GSa/s (2ch), 500 MSa/s (3-4ch) |
| Memory | 2.5 kpts per channel |
| Resolution | 8-bit |
| Display | Color TFT LCD |

## TDS2CMA Module

The communications module adds:
- FFT math
- GPIB interface
- RS-232 interface
- Centronics parallel printer port

**Note:** This scope has NO USB and NO CompactFlash storage.

## Features

- Edge/video/pulse width triggers
- 11 auto measurements
- Roll mode
- Average mode
- Dual timebase
