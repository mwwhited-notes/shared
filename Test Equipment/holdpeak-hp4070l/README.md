# Holdpeak HP-4070L

**Handheld LCR Meter**

## Specifications

| Specification | Value |
|--------------|-------|
| Display | 3½ digit (1999 counts) |
| Resistance | 20 Ω to 2000 MΩ (9 ranges) |
| Capacitance | 2000 pF to 200 µF (6 ranges) |
| Inductance | 20 mH to 20 H (4 ranges, 100 Hz test) |
| Dimensions | 143 × 75 × 32 mm |
| Power | 9V battery |

## Features

- Transistor hFE test
- Backlight
- Auto power-off
- Manual ranging

## Notes

Basic LCR meter for quick component checks. For more precise measurements at multiple frequencies, use the DE-5000.
