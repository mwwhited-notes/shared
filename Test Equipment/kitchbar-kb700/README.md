## Features

- Image rotation (up/down, left/right)
- Image mirroring
- 10-level brightness adjustment
- Auto focus
- Flexible gooseneck arm
- Plug and play (no drivers needed)

## Compatibility

- Windows 7/8/10/11
- macOS 10.11+
- Chrome OS
- Works with Zoom, OBS, etc.

## Applications

- PCB inspection and documentation
- Component photography
- Work-in-progress documentation
- Macro photography of small parts
- Live streaming bench work
- Before/after repair photos
