# QE-MS305D (QW-MS305D)

**Adjustable DC Power Supply**

## Specifications

| Specification | Value |
|--------------|-------|
| Type | Linear regulated |
| Output | 0-30 V, 0-5 A |
| Display | 3-digit LED |
| Load Regulation | CV ≤0.1% + 5 mV, CC ≤0.1% + 10 mA |
| Ripple | CV ≤10 mVrms, CC ≤20 mArms |
| Input | 110 V AC |

## Features

- 4 potentiometers (coarse/fine voltage and current)
- CV/CC auto-switching

## Notes

Basic manual bench supply. For programmable control, use the Korad KA3005P.
