## Features

- Simul-focal trinocular head (camera port usable with both eyepieces)
- Darkfield illumination for enhanced contrast
- Compensation-free Siedentopf head design
- Precision focus with tension adjustment and vertical lock
- Low-position stage controls for ergonomics
- C-mount compatible trinocular port

## Applications

- Blood work and fluid analysis
- Pathology
- Veterinary diagnostics
- Cell biology
- Microbiology
