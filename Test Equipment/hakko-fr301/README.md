- Standalone operation (no separate station required)
- Built-in vacuum pump
- Quick-change N61 nozzle system
- High thermal efficiency nozzles
- Reduced solder clogging design
- One-hand operation
- Replaceable filters

| Nozzle | Size | Type | Use Case |
|--------|------|------|----------|
| Stock | 1.0mm | Standard | General through-hole |
| N61-16 | 1.0mm (4mm body) | Standard | General through-hole |
| N61-13 | 1.3mm | Extra long | Tight spaces, recessed joints |

## N61 Nozzle Compatibility

The FR-301 uses N61 series nozzles (also compatible with FR-410x series):
- Sizes: 0.6mm to 1.6mm diameter
- Shapes: Round and oval available
- Note: N61 nozzles are NOT compatible with older FR-300

## Maintenance

- Clean filter regularly (check after each session)
- Clear nozzle if solder accumulates
- Replace cleaning pipe as needed
- Keep vacuum path clear
