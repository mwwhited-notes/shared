# Fifine AmpliGame D6 Stream Controller

**Programmable Macro Keypad**

## Unit Information

| Field | Value |
|-------|-------|
| Manufacturer | Fifine |
| Model | AmpliGame D6 |
| Type | Macro Keypad / Stream Controller |
| Condition | Working |

## Specifications

| Specification | Value |
|--------------|-------|
| Keys | 15 programmable macro keys |
| Compatibility | Windows, macOS |
| Interface | USB |
| Software Support | OBS, Twitch, YouTube, Streamlabs |

## Features

- 15 fully programmable macro keys
- Custom trigger actions per key
- Cross-platform (Mac/PC)
- Integration with streaming software
- Dedicated keypad form factor

## Applications

- **Instrument Control**: Quick access to common SCPI commands
- **Test Automation**: Trigger measurement sequences
- **Documentation**: One-click screen capture, video start/stop
- **Software Shortcuts**: Launch oscilloscope, logic analyzer software
- **Streaming**: Scene transitions for project streams

## Macro Ideas for Electronics Work

| Key | Function |
