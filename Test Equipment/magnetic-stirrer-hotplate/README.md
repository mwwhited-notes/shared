# Magnetic Stirrer Hot Plate

**Laboratory Magnetic Stirrer with Hot Plate**

## Unit Information

| Field | Value |
|-------|-------|
| Type | Magnetic Stirrer / Hot Plate |
| Condition | Working |

## Features

- Combined magnetic stirring and heating
- Temperature controlled hot plate
- Variable stirring speed
- Suitable for liquid mixing applications

## Applications

- Chemical solution mixing
- PCB flux activation
- Solder paste mixing
- Cleaning solution preparation
- Resin/epoxy mixing

## Safety Notes

- Allow hot plate to cool before handling
- Use appropriate magnetic stir bars for container size
- Do not heat flammable solvents
- Keep away from sensitive electronics when stirring
