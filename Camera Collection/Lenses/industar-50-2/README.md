## Features

- Ultra-compact "pancake" design
- Tessar-derived 4-element optical formula
- Nicknamed "Eagle's Eye"
- Clickless aperture (good for video)
- Produced for Zenit SLR cameras
- Minimal front coating, rear anti-reflective coating

## Compatible Bodies

- Native: M42 screw mount cameras (Mamiya, Pentax, Zenit)
- Adapted: Nikon F, Canon EF, Sony E, Fuji X, Micro 4/3
