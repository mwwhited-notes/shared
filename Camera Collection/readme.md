## Mount Compatibility

### Adapting M42 Lenses

M42 screw-mount lenses can be adapted to:
- **Nikon F-mount** - With M42-Nikon adapter (infinity focus maintained with correction element)
- **Canon EF** - With simple adapter
- **Sony E** - With simple adapter
- **Fuji X** - With simple adapter
- **Micro Four Thirds** - With simple adapter

### Nikon F-Mount

Native F-mount lenses work on:
- Nikon D800 (FX, full metering/AF with AF-S lenses)
- Nikon FG-20 (AI metering, manual focus)
