## Features

- First SLR with dual metering system (spot + averaging)
- M42 universal screw mount (vast lens compatibility)
- Introduced January 1968
- Variable delay self-timer
- Metal body construction (black and silver versions)
- "1000" indicates top shutter speed of 1/1000s

## Compatible Lenses

- All M42 screw-mount lenses:
  - Mamiya Sekor M42 lenses
  - Pentax Super-Takumar
  - Carl Zeiss Jena
  - Helios (Soviet)
  - Industar (Soviet)
  - Many others
