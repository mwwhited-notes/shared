## Specifications

### PCH-1000 (OLED Model)

| Specification | Value |
|--------------|-------|
| SoC | ARM Cortex-A9 MPCore (4 cores) |
| CPU Speed | 444 MHz (2 GHz capable, locked) |
| GPU | SGX543MP4+ |
| RAM | 512 MB |
| VRAM | 128 MB |
| Storage | Internal (1GB) + Proprietary memory card |
| Display | 5" 960x544 OLED |
| Connectivity | WiFi, Bluetooth, 3G (optional) |
| Ports | Proprietary USB, memory card slot |
| Battery | 2210 mAh |
| OS (Stock) | PlayStation Vita System Software |

### PCH-2000 (LCD/Slim Model)

- LCD screen instead of OLED
- 1 GB internal storage
- Micro-USB charging
- Slimmer, lighter

### PS TV (PlayStation TV / Vita TV)

- No screen (TV output via HDMI)
- Uses DualShock 3/4
- Ethernet port
- Some games have compatibility issues

### Available Modifications

| Modification | Availability | Notes |
|-------------|--------------|-------|
| HENkaku | 3.60-3.65 | Best firmware for homebrew |
| Enso | 3.60-3.65 | Permanent CFW (survives reboot) |
| h-encore² | 3.65-3.74 | For newer firmware |
| Trinity | 3.69-3.74 | Alternative exploit |
| SD2Vita | Hardware adapter | Use microSD instead of Sony cards |
| PKGj | Homebrew app | Game backups |
| Adrenaline | Homebrew app | Full PSP emulator |
| RetroArch | Homebrew app | Multi-system emulation |

### Recommended Firmware

- **3.60** - Most compatible, best for homebrew
- **3.65** - Also excellent, Enso available
- Higher firmware can be downgraded using modoru

### SD2Vita

Highly recommended mod - allows use of cheap microSD cards instead of expensive proprietary Sony memory cards. Requires:
- SD2Vita adapter (cheap, ~$5)
- microSD card
- CFW installed first
