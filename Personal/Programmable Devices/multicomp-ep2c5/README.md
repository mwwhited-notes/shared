## Features

- Grant Searle Multicomp design
- 512 KB banked SRAM for CP/M
- VGA video output (accent color accent scheme)
- PS/2 keyboard input
- SD card for storage
- USB serial console
- Extra I/O for expansion
- Open source design

## Software/HDL

- **Quartus 13.1 SP** - Required for Cyclone II
- **Multicomp HDL** - Grant Searle's design
- Various CPU configurations available

## Applications

- CP/M computing
- 6502 retrocomputing
- Learning FPGA development
- Running vintage software
- Educational projects

## Required Components (Not Included)

- EP2C5 FPGA module (~$12 on AliExpress)
- VGA connector
- PS/2 connector
- FTDI USB-serial
- SD card socket
- SRAM chips
- Passive components
