## Features

- Soviet copy of Carl Zeiss Biotar 58mm f/2
- Famous "swirly bokeh" effect
- Multi-coated version (better flare resistance than earlier versions)
- All-metal construction
- Classic 6-blade aperture

## Compatible Bodies

- Native: M42 screw mount cameras (Mamiya, Pentax, Praktica)
- Adapted: Nikon F (with M42-Nikon adapter, infinity focus possible)
- Adapted: Canon EF, Sony E, Fuji X, Micro 4/3 (with appropriate adapters)
