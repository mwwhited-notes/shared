## Features

- Nikon's last and most advanced professional 35mm SLR
- 11-point autofocus with 3D tracking
- 1005-pixel RGB metering sensor
- Built-in motor drive (5.5 fps standard, 8 fps with MB-40)
- Weather-sealed magnesium alloy body
- DX lens coding recognition
- Multiple exposure capability
- Mirror lock-up
- Depth-of-field preview
- Produced 2004-2020 (one of the last film SLRs ever made)

## Compatible Lenses

- All Nikon F-mount lenses:
  - AF-S (full autofocus)
  - AF-D (full autofocus)
  - AF (full autofocus)
  - AI-S, AI (metering, manual focus)
- G-type lenses (fully supported)
