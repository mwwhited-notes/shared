## Features

- Blue LED output level display
- USB MP3 playback from flash drive (up to 32GB)
- Microphone input with independent volume
- Switchable AUX1/AUX2/CD/USB input selection
- Microphone, pager, and mixing modes
- Bass and treble tone controls
- Toroidal transformer for cleaner power
- Compact form factor
