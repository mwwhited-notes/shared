## How to Use This File

### When to Add Entries

Add an entry when:
- Download fails after trying all fallback methods
- Site requires login/registration
- Document not available online
- Downloaded file was wrong content

### Entry Format

```markdown
### Board Name
- [ ] **Document Type** - Description
  - Source: [best URL]
  - Issue: [why it failed]
  - Manual download: [instructions]
```

### When You Obtain a Document

1. Place file in the correct board directory
2. Update the board README's "Local Files" section
3. Move the entry to "Completed" section below
