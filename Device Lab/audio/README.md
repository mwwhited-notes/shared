## SOLUPEAK NA2 Phono Preamp

Moving Magnet (MM) phono preamplifier for turntable connection.

### Specifications

| Specification | Value |
|--------------|-------|
| Type | Phono Stage Preamplifier |
| Cartridge Type | MM (Moving Magnet) only |
| Input | RCA (from turntable) |
| Output | RCA line level |
| Power | USB or DC adapter |

### Use Case
- Connects turntable to line-level inputs (Sonos Port, receivers without phono input)
- Provides RIAA equalization for vinyl playback
