## Computing Elements

| Element | Quantity | Function |
|---------|----------|----------|
| Integrators | 5 | Time integration (∫) |
| Summers | 4 | Addition/subtraction |
| Multipliers | 2 | Analog multiplication |
| Comparators | 2 | Digital comparison |
| Coefficients | 8 | Variable gain (potentiometers) |
| Inverters | Several | Sign inversion (-1×) |

## Features

- Open-source hardware (schematics on GitHub)
- Daisy-chain interface for connecting multiple THATs
- Digital panel meter for precision measurements
- Hybrid interface for analog-digital programming
- 2mm patch cable system
- Manual controls for hands-on operation

## Applications

- Differential equation solving
- Dynamic systems simulation
- Control system modeling
- Physics simulations (pendulum, oscillators, chaos)
- Educational demonstrations
- Hybrid analog-digital computing
